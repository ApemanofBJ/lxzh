[安装]

cd ~/.vim/bundle
git clone https://github.commattn/emmet-vim.git
把 plugin/emmet.vim 拷贝至 Vim 安装目录下的 plugin/ 文件夹。
将 autoload/ 下的 emmet.vim 和 emmet/ 文件夹拷贝至 Vim 安装目录下的 autoload/ 文件夹

[使用] 官方文档

快捷键： <c-y>,
# 修改快捷键： let g:user_emmet_expandabbr_key = '<F2>'

HTML 規則

    基本规则：
  • E 标签
  • E*n -> n个标签
  • E#id 标签有id属性
  • E.class 标签有class属性
  • E>N N是E的下级元素
  • E+N N是E的同级元素
  • E^N N是E的上级元素
  • E{info} E包含的内容info
  • E[属性="value"] 设置属性
a.) 建立HTML文件 html:5 -> 快捷键 b.) 缩写 a:link -> $ -> 变量自增加 a:mail -> img -> c -> 注释 c.) 标签 选择文本 -> 按'V' -> 快捷键 -> 输入标签 ul{} [实例] 1. h${这是一个标题}*3
