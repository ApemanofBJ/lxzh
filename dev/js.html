二、DOM & 对象

Document Object Model-文档对象模型，操作对象的属性、内容、样式

§2.1 document 对象

获取元素
  • document.getElementById("id")
  • document.getElementsByTagname("标签")
  • document.getElementsByName("name")
改变元素样式 document.getElementById(标签).style.样式="新值" #去掉-,后单词首字母大写
改变元素的内容
  • document.innerText #文本-IE
  • document.textContent #非IE
  • document.innerHtml #带标签
显示和隐藏（display属性） Object.style.display = none 隐藏/display 显示

表单对象.forms[], 其属性值是一个文档中所有form元素对象的数组。或者，使用表单名称作为索引。
设置属性：name、target、action、methon和enctype。
.images[] 是标记img 对象的数组集合。使用value取对应文本
