第一章 基础

光标的移动 方向键
  • h--head 左
  • l--last 右
  • k--korea 上
  • j--japan 下
行首、尾部
  • ^ 行首字符(或者 0-行首)/$ 行尾字符
单词移动
  • w 单词，包含符号和标点(word)
  • W 单词，不包含符号和标点
  • e 词尾(end)
  • E 词组尾部
  • b 单词回退(back )
  • word--以非空白符分割的单词/WORD--以空白符分割的单词
更改 单词更改
  • cw 向后更改单词
  • cb 向前更改单词
  • c$ 更改到行尾
  • c0 更改到行首
  • cc 更改整行(或者C)
  • r 更改单个字符
  • R 连续更改字符
  • c#h/l 向前/后修改#个字符
  • s 替换并进入插入模式
  • 文本对象：单词w/句子s/段落p
  • i(/i</i{--操作括号内容
大小写更改
  • ~ 单个字符更改大小写
  • guw 单词转换大小写 #U 代表大写
  • guu 至行尾转换小写 #gUU 转换未大写
  • guG 至文件尾部转换大小写
  • gu# 行转换大小写
删除 删除内容(保存到寄存器)
  • dd 删除整行
  • D 删除至行尾
  • x 删除字符
  • xp 字符对调
  • u 恢复 #U 恢复整行操作
  • Ctrl+h 删除上一个字符/Ctrl+w 删除上一个单词/Ctrl+u 删除整行
复制 复制内容-yanking
  • yy 复制整行
  • yw 复制单词
  • p 向下粘贴
  • P 向上粘贴
  • y$ 复制到行尾
  • y0 复制到行首
  • #yh/l 复制连续字符
插入 插入文本
  • A 行尾插入
  • I 行首插入
  • o 下行插入
  • O 上行插入
  • S 删除整行替换文本
  • s 删除字符替换文本
  • i--insert/a--append/o--open a line below
    备注：
  1. 位于单词中间时删除单词：daw #delete a word
  2. 修改句子：cis #change inner sentence
  3. . 重复命令

第五章 全局替换

格式： s/old/new/ #substitute
即[range]s[ubstitute]/{pattern}/{string}/[flags]
flags:g 全部替换/c 替换确认/n 报告匹配的次数而不进行替换(用来查询匹配次数)

设置查找单词： * '#'反向查找

§5.1 搜索替换

g/搜索文本/s/old/new/ 对于搜索到的行进行文本的替换

    备注
  • / 原义
  • \(文本\) 使用\1-9替代 #即反向引用
  • /<\> 开头\结尾
  • ~ 匹配上次正则表达式
  • & 替换字符串
  • \u 替换大小写
  • /l 小写字母

搜索选项
set ic 忽略大小写
set noic 分辨大小写

example:

  • 清除尾部空格 ss/+$//
  • 加入tab s/^t/
  • 删除前#个字符 s/^./#/
  • 替换英文 s/[a-zA-Z]/+/&字符串/
  • 匹配汉字 [/u4e00-u9fa5]
  • 匹配重复的第#项 !sed 's/匹配文本/替换文本/#'
  • 单词首字母大写 s/\s\w/\U&/
  • 删除空行 g/^$/d
  • 清除HTML标签 s/<\/\?p>//g
  • 原意替换 []|
  • 统计当前行文本次数 %s/pattern//gn

§5.2 global 命令

说明：range 默认是整个文件/未匹配进行操作--global!或者vglobal

§6.4 map 映射

格式： map new old
模式： nmap/vmap/imap #命令、列操作、插入
note: nore--no recurse 无递归

    快捷键：
  • <ESC> 代表Escape键
  • <CR> Enter键
  • <C-ESC> 代表Ctrl + ESC
  • <S-F1> 表示Shift + F1
  • <A-key> 表示Alt + key
  • <FX> 表示FN功能键
  • <LEFT> 表示方向键
    ex:
  • :map <F5> i<Esc>ea<Esc> #F5键映射为给word加上
  • :map /p i<Esc>ea<Esc> #/p键映射为给word加上
  • :map : 映射空格
  • imap A 在插入模式下，光标到最后

备注：
:map #列出所有映射

§8.1 Vundle --豌豆

安装： git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

配置：.vimrc

set nocompatible "required
filetype off "required!

set rtp+=~/.vim/bundle/vundle/
call vundle#begin()
Plugin '' call vundle#end()
filetype plugin indent on " required
Bundle 'gmarik/vundle'

    指定插件来源：
  • a. 指定Github中vim-script仓库中的插件，直接指定名称即可
  • b. 指定Github中其他用户仓库插件，指定方式：用户名/插件名称
  • c. 指定非Github中的git仓库的插件，需要用git地址
  • d. 本地git中的插件

安装插件：PluninInstall
卸载插件：PluginClean, #在文档中删除插件行，保存退出后再运行此命令
搜索插件：PluginSearch 插件名称
