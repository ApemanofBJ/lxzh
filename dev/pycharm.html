    设置-Ctrl + Alt + S
  • vim插件: setting --> plugin --> IdeaVim
  • 安装库: setting --> project --> project interpreter --> 查找库并进行安装
  • 解释器：setting-->python-->project interpreter-->设置：Existing environme...
  • 查看运行期变量值：Edit Configurations-->Show command line afterwards
功能 快捷键

Shift + F10 运行当前程序
当前行上移/下移 Ctrl + Shift + Up/Down
关闭其他标签页 选择当前页签，Alt+鼠标单击关闭按钮
