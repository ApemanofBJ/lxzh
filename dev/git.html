git

原理：基于版本快照工作，保证数据完整性(所有数据在存储前都计算校验和，然后以校验和来引用)
管理文件夹分为三个区域：工作区、暂存区、分支区。

git -基本操作

初始化命令： git init, 会在当前目录建立一个.git目录进行记录,用来跟踪管理版本库
设置 用户名：git config --global suer.name “设置的名字”
邮箱： git config --global user.email “邮箱地址”
查看配置信息：git config --list
添加文件快照：(跟踪文件) git add 目录或文件名称,#添加所有文件: git add -A/--all
提交暂存区的文件 git commit -m
状态查看： git status
查看文件修改记录： git log -p filename

GitHub--是世界上最大的代码托管平台

备注：查询登录信息

加入SSH-key

  1. 生成key-pair: ssh-keygen -t rsa -b 4096 -C "name@github.com"
    默认文件名称：/home/lxzh/.ssh/id_rsa、id_rsa.pub
  2. ssh-agent: eval "ssh-agent -s"; ssh-add ~/.ssh/id_rsa
  3. Add account: apt install xclip; xclip -sel clip < ~/.ssh/id_rsa.pub
  4. 连接：ssh -T git@github.com
