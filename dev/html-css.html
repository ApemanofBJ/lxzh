第二部分 CSS-层叠样式表

属性设置参考手册

	
	/* …… */ 注释语句
    使用方式
  1. 内联方式：style="属性列表"
  2. 嵌入方式: 在head 部分加入 style进行设置
  3. 外部样式表: 单独定义为.css文件，通过link 进行连接
    link rel="stylesheet" href="../../CSS/global.css"
  4. 输入样式表: @import url(在.css文件中引入其他的css文件格式)
    选择器
  • 派生选择器：通过父标签 子标签的方式修改样式,即包含关系
  • id选择器：通过#id的方式修改样式
  • 类选择器: 通过.class的方式修改样式
  • 属性选择器:通过[属性="值"]的方式修改样式

div标签：是块级元素，可用于布局(作为容器使用)。#division 分区

display属性
  • block 默认值，容器中所有元素被当做一个独立的块放置到页面中；
  • none 隐藏对象

span标签：是内联元素，用作文本容器。#对包含的文本可以进行CSS设置

border-width支持关键字：thin(1px)、medium(3px)、thick(5px)
