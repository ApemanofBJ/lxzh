开机取消PIN码登录
  1. 运行：netplwiz
  2. 取消勾选：“需要密码选项”
  3. 输入确认的PIN密码
浏览器问题 Firefox 不支持H5视频播放
  1. 安装插件：User Agent Switcher：
    Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/34.0.1847.137 Safari/537.36
  2. 生效：打开网页，摁一下Alt 键，选择工具，选择所添加的那个即可
360浏览器设置内核
  1. 选择设置-&glt;高级设置-&glt;内核设置
  2. 选择OA访问方式:webkit内核方式
苹果电脑问题 蓝牙鼠标开机不识别
  1. 手机连接蓝牙鼠标，确定鼠标是否好坏
  2. 进入电脑系统，选择蓝牙设备，删除鼠标"Magic Mouse 2"，重新建立配对连接
打印机问题 Win10系统正常安装后，无法打印
  1. 在管理中检查printspool服务是否启动
  2. 安装驱动精灵程序，检查本地打印服务
  3. 在system32/spool/driver文件夹下删除相关驱动后重新安装即可
