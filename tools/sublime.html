§chapter2 应用技巧

§2.1 日常

  • 对频繁打开进行编辑的文件,可以打开它的上层文件夹来进行定位
  • 外部程序修改文件内容后自动更新
  • Ctrl+P, 输入：可以跳转行/输入@，跳转到定义位置/输入#，跳转到字符串
  • 快速更改所有同名内容:使用Alt + F3
    多行游标：Ctrl + D, 跳过 Ctrl + K
  • 使用vi模式：在"首选项preference"-->"设置settings-用户user"-->"ignored_packages"内容设置为空(//注释掉即可)
  • 自动识别文件类型，自动保存临时文件，且可以直接打开图片格式文件
  • Ctrl+Shift+p(命令模式)，设置文件格式
  • 代码格式化，Ctrl+Shift+V
  • snippets: 使用tab 进行输入跳转
  • $file, 表示当前文件

§2.2 高级

插入当前日期

1. 创建插件：Tools → New Plugin:

import datetime import sublime_plugin class AddCurrentTimeCommand(sublime_plugin.TextCommand): def run(self, edit): self.view.run_command("insert_snippet", { "contents": "%s" % datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S") } )

2. 创建快捷键：Preference → Key Bindings - User

[ { "command": "add_current_time", "keys": [ "ctrl+shift+." ] } ]
