step 1: 绘制函数图形

命令: plot [-pi:pi] cos(x),sin(x),…… #函数列表
用分号;分割多个命令

设置
命令 说明
set title "title" 设置图形名称
set xlabel(xl) "X" 设置x轴标签
set ylabel(yl) "Y" 设置y轴标签
set xrange [-pi:pi] 设置x轴的坐标范围
set xdata time 设置X轴为时间
set xtics min,step,max 设置主刻度
set mxtics number 设置分刻度
set sample number 设置取样数目，默认是100个
set term wxt background rgb "gray75" 设置背景颜色
set grid 显示虚线网格
set terminal png 设置输出图片为png格式
set output "output.png" 设置输出图片的文件名

step 2: .dat数据文件绘图

§2.1 点线图

命令：plot "file.dat" title "图形名称"
using 数据列(m:n) #通过这种方式支持多数据绘图(列顺序通过：分开),缩写u

with 参数
参数 说明
linestyle--连线风格 set linestyle 1 1w 2c rgb "blue" pointtype 6 #线类型1--蓝色
set linestyle 2 2w 2 lc rgb "forest-green" pointtype 8 #线类型2--绿色
linetype--连线种类
  • 0-black
  • 1-red
  • 2-green
  • 3-blue
  • 4-pink
  • 6-yellow
linewidth--连线粗细 缩写lw, 值1-6
linecolor--连线颜色 缩写lc,
pointtype--点的种类 缩写pt,
pointsize--点的大小 缩写ps,
