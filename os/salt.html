服务架构：主端通信需加密
  • master: 负责管理所有节点
  • minios: 终端节点服务
  • zeroMA: 通信服务, #替代 RAET
  • AES: 数据加密方法

安装:
yum install salt-master salt-minion

启动: /etc/init.d/salt-master start

    配置文件
  • interface: 监听的IPv4的地址
  • PUBLISH_PORT: 通信端口

测试: salt minion-id test.ping

salt-key
  • -L 列出终端节点
  • -a 节点名称 #添加节点
